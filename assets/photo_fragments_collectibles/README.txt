Collectible memory photo fragments for the game.

Theme:
- Level 1: hospital memory
- Level 2: adulthood memory
- Level 3: childhood memory

Recommended use:
- Place 3 fragments in each level.
- Each fragment PNG is 64x64 with transparent background and glow.
- When all 3 fragments of a level are collected, show the corresponding complete preview image.
- The complete previews are for UI / ending / cutscene screens.
